# tic-tac-toe

A tiny responsive **static** Tic‑Tac‑Toe web app.

Features:
- Responsive 3×3 grid UI
- Win + draw detection (highlights winning line)
- Restart button
- Optional single-player mode vs AI (minimax)

## Run

### Option A — open directly
Just open `index.html` in a browser.

### Option B — serve locally
From the project folder:

```bash
npx serve
```

Then open the URL printed by `serve`.

## Files

- `index.html` — markup
- `style.css` — styling
- `app.js` — game logic + AI
